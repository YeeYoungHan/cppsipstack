===========
SipLoadBalancer
===========

1. 개요

 * C++ SIP stack 을 이용한 SIP 로드밸런서 개발
 
 * 목적
   - C++ SIP stack 을 이용한 SIP 로드밸런서 개발

4. 기능 설명

 Sip 로드밸런서 0.10 버전의 주요 기능은 다음과 같습니다. 

 * 설정 기능 
   - XML 파일에 설정 사항을 설정하는 기능 

 * IP-PBX 로드밸런싱 기능 
   - XML 파일에서 연동할 IP-PBX 정보를 가져오는 기능 

 * 모니터링 기능 
   - 내부 자료구조에 대한 모니터링 기능 ( SipLoadBalancerMonitor.exe 로 모니터링 할 수 있음 )
