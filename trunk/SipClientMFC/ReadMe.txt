===========
SipClientMFC
===========

1. 개요

 * C++ SIP stack / SipUserAgent 를 이용한 MFC 기반 SIP 클라이언트 개발
 
 * 목적
   - C++ SIP User Agent 를 이용하여서 MFC 기반 SIP 클라이언트를 개발할 수 있는지 보여주기 위한 예제 프로그램
   - C++ SIP User Agent 를 사용하는 방법을 알려주기 위한 예제 프로그램

 * 주요 기능
   - 로그인 기능
   - 통화 시작/취소/종료 기능
