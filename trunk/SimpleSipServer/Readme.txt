===========
SimpleSipServer
===========

1. 개요

 * C++ SIP stack 을 이용한 초간단 SIP 서버 개발
 
 * 목적
   - C++ SIP stack 을 이용하여서 얼마나 쉽게 SIP 서버를 개발할 수 있는지 보여주기 위한 예제 프로그램
   - C++ SIP stack 을 사용하는 방법을 알려주기 위한 예제 프로그램

 * 주요 기능
   - SIP REGISTER 수신시 사용자 자료구조에 저장하는 기능
   - SIP INVITE / CANCEL / ACK / BYE 전달 기능
