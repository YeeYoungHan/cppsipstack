=================
SipLoadBalancerMonitor
=================

1. 개요

 * SipLoadBalancer 의 내부 자료구조를 모니터링하기 위한 XML 개발
 
2. 개발자 정보
  
 * 이영한
 * 이메일 : websearch@naver.com
 * 블로그 : http://blog.naver.com/websearch

3. 라이선스

 * 본 프로젝트의 라이선스는 GPLv3 이고 기업용 라이선스는 websearch@naver.com 으로 문의해 주세요.

4. 사용 방법

 * ServerMonitor 프로그램을 SipLoadBalancer 모니터링을 위해서 설치한다.
 * ServerMonitor 프로그램이 설치된 폴더에 ServerMonitor.xml 를 복사한다.
 * ServerMonitor 프로그램을 실행한다.

5. 기타

 * 2014년 4월 9일에 SipLoadBalancerMonitor 프로젝트를 삭제함
 * ServerMonitor 프로그램이 SipLoadBalancerMonitor 프로그램의 모든 기능을 수행함
