===========
McuEmulator
===========

1. 개요

 * C++ SIP stack 을 이용한 Radvision MCU Emulator
 
 * 목적
   - C++ SIP stack 을 이용하여서 Radvision MCU 의 SIP 프로토콜과 유사한 기능을 수행한다.
   - Radvision MCU 의 음성/영상 처리 기능은 개발하지 않고 단순히 SIP 클라이언트에서 수신한 RTP 를 relay 하는 기능으로 개발한다.

 * 주요 기능
   - SIP INVITE / UPDATE / INFO / ACK / BYE 수신 기능
   - 회의방에 대한 RTP relay 기능
