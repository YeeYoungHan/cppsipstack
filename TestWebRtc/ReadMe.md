### 실행 방법
TestWebRtc.exe 프로그램은 TestWebRtc 폴더의 html 폴더를 document root 로 설정하여 실행하면 됩니다.

```
# TestHttpStack.exe {local ip} C:\OpenProject\CppSipStack\trunk\TestWebRtc\html
```
