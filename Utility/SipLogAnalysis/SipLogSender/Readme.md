# SipLogSender

### 개요

* SIP 서버에서 저장한 로그 파일을 읽어서 로그 파일에 저장된 SIP 메시지를 SIP 서버로 전송하는 프로그램
 
### 개발자 정보
  
* 이메일 : websearch@naver.com
* 블로그 : http://blog.naver.com/websearch

###

* 사용법

'''
SipLogSender.exe {listen udp port} {sip server udp port} {log filepath} 
'''
