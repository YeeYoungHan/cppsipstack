# SipLogReassembly

### 개요

* SIP 로그 파일을 읽어서 입력된 전화번호들에 대한 SIP 로그를 신규 로그 파일에 저장하는 프로그램
 
### 개발자 정보
  
* 이메일 : websearch@naver.com
* 블로그 : http://blog.naver.com/websearch

###

* 사용법

'''
SipLogReassembly.exe {log filepath} {output filepath} {phone} ...
'''
