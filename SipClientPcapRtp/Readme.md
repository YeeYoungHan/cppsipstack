# SipClientPcapRtp

### 개요

* C++ SIP stack / SipUserAgent 을 이용하여서 통화 연결한 후, pcap 파일에 존재하는 RTP 패킷을 전송하는 SIP 클라이언트

### 주요 기능

* 로그인 기능
* 통화 연결되면 pcap 파일에 존재하는 RTP 패킷을 전송한다.
* XML 설정 파일을 읽어서 pcap 파일 및 pcap 파일에서 읽을 RTP 목적지 IP 주소/포트 번호를 설정한다.
